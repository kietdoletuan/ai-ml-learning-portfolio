# ai-ml-learning-portfolio

Grade 11 student from Vietnam building AI/ML projects alongside a structured AI/DS course.  
---

## Repository Structure

```
ai-ml-learning-portfolio/
│
├── foundations/          - Python core, NumPy, Pandas, data viz, SQL basics
│
├── data_analysis/        - Excel, Power Query, Power BI, DAX, Tableau
│
├── classical_ml/         - KNN, Decision Trees, Ensemble Methods, Regression
│
├── deep_learning/        - PyTorch, CNN, RNN/LSTM, Transformers
│
├── mlops/                - FastAPI, model deployment, serving
│
├── projects/             - Independent projects not tied to a single lesson
│
├── kaggle/               - Competition notebooks and submissions
│
└── capstone/             - End-of-year major project
```

Each lesson subfolder follows this layout:

```
topic_domain/
└── lesson_name/
    ├── lesson_replication.py    - code from the lesson itself
    ├── project_[name].py        - independent build using the lesson concept
    ├── data/                    - datasets used (CSV, XLSX, etc.)
    └── README.md
```
